# data-cleaning-and-preprocessing-
Overview

This project contains a ready-made Excel file designed to demonstrate basic data-cleaning operations such as:

Handling missing values

Removing duplicates

Standardizing text

Cleaning dates

Fixing data types

Producing a final cleaned dataset

The workbook includes:

RawData – uncleaned, messy data

CleanedData – fully cleaned version 




Usage

You can use this file for:

Data analysis practice

Learning data cleaning techniques

Training for Excel/Python data preprocessing tasks

Assignments and portfolio projects
